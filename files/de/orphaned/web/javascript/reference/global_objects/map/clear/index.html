{{JSRef}}

Die clear() Methode entfernt alle Elemente von einem Map Objekt.

{{EmbedInteractiveExample("pages/js/map-prototype-clear.html")}}

Syntax

myMap.clear();

Rückgabewert

{{jsxref("undefined")}}.

Beispiele

Einsatz der clear Methode

var myMap = new Map();
myMap.set('bar', 'baz');
myMap.set(1, 'foo');

myMap.size;       // 2
myMap.has('bar'); // true

myMap.clear();

myMap.size;       // 0
myMap.has('bar')  // false
