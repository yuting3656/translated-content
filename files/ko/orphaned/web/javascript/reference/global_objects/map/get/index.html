get() 메서드는 Map 객체에서 지정한 요소를 회수합니다.

{{EmbedInteractiveExample("pages/js/map-prototype-get.html")}}

구문

myMap.get(key);

매개변수

key
Map 객체에서 회수할 요소의 키.

반환 값

주어진 키와 연결된 요소. 그런 요소가 없으면 {{jsxref("undefined")}}.

예제

get 사용하기

var myMap = new Map();
myMap.set('bar', 'foo');

myMap.get('bar');  // "foo" 반환.
myMap.get('baz');  // undefined 반환.
