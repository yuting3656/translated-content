has() 메서드는 주어진 키를 가진 요소가 Map에 존재하는지를 반환합니다.

{{EmbedInteractiveExample("pages/js/map-prototype-has.html")}}

구문

myMap.has(key);

매개변수

key
존재 여부를 판별할 키값.

반환 값

주어진 키를 가진 요소가 있으면 true, 아니면 false.

예제

has() 사용하기

var myMap = new Map();
myMap.set('bar', "foo");

myMap.has('bar');  // returns true
myMap.has('baz');  // returns false
