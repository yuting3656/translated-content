Map() 생성자는 {{jsxref("Map")}} 객체를 생성합니다.

구문

new Map([iterable])

매개변수

iterable
요소가 키-값 쌍인, {{jsxref("Array")}} 또는 다른 순회 가능한 객체(예: [[1, 'one'], [2, 'two']]). 각 키-값 쌍은 새로운 Map에 포함됩니다.

예제

let myMap = new Map([
  [1, 'one'],
  [2, 'two'],
  [3, 'three'],
])
