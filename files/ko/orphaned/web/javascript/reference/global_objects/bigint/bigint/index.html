BigInt() 생성자는 {{jsxref("BigInt")}} 객체를 생성합니다.

구분

BigInt(value);

매개변수

value
생성하는 객체에 할당할 숫자 값. 문자열 또는 정수를 사용할 수 있습니다.

참고: BigInt()는 {{jsxref("Operators/new", "new")}} 연산자와 함께 사용하지 않습니다.

예제

BigInt(123);
// 123n
