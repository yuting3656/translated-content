Description

La valeur de size est un entier représentant le nombre d'entrées d'un objet Map. Le mutateur correspond à cette propriété est {{jsxref("undefined")}}, on ne peut pas donc pas modifier cette propriété.

Exemple

Utiliser size

var maMap = new Map();
maMap.set("a", "alpha");
maMap.set("b", "beta");
maMap.set("g", "gamma");

maMap.size // 3
