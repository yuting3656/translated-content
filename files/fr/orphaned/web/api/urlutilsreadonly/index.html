{{ApiRef("URL API")}}{{SeeCompatTable}}

L'interface URLUtilsReadOnly définie les méthodes utilitaires permettant de travailler sur les URLs. Cette dernière définie seulement les méthodes sans modifier le contenu dont l'usage se fait sur des données qui ne peuvent être changées.

Il n'existe aucun objet de ce type, mais certaines interfaces l'implémente, comme {{domxref("WorkerLocation")}}.

Propriétés

Cette interface n'hérite d'aucune propriété.

{{domxref("URLUtilsReadOnly.href")}} {{readOnlyInline}}
Un {{domxref("DOMString")}} contenant l'URL entier.
{{domxref("URLUtilsReadOnly.protocol")}} {{readOnlyInline}}
Un {{domxref("DOMString")}} contenant le schéma protocolaire de l'URL, incluant le caractère ':' à la fin.
{{domxref("URLUtilsReadOnly.host")}} {{readOnlyInline}}
Un {{domxref("DOMString")}} contenant le hôte, qui est le nom d'hôte, un caractère ':', et le port de l'URL.
{{domxref("URLUtilsReadOnly.hostname")}} {{readOnlyInline}}
Un {{domxref("DOMString")}} contenant le domaine de l'URL.
{{domxref("URLUtilsReadOnly.origin")}} {{readOnlyInline}}
Retourne un {{domxref("DOMString")}} contenant sous la forme canonique l'origine de l'emplacement spécifique.
{{domxref("URLUtilsReadOnly.port")}} {{readOnlyInline}}
Un {{domxref("DOMString")}} contenant le numéro du port de l'URL.
{{domxref("URLUtilsReadOnly.pathname")}} {{readOnlyInline}}
Un {{domxref("DOMString")}} contenant au début '/' suivi du chemin de l'URL.
{{domxref("URLUtilsReadOnly.search")}} {{readOnlyInline}}
Un {{domxref("DOMString")}} contenant le caractère '?' suivi par les paramètres de l'URL.
{{domxref("URLUtilsReadOnly.hash")}} {{readOnlyInline}}
Un {{domxref("DOMString")}} contenant le caractère '#' suivi par le fragement d'identification de l'URL.

Méthodes

Cette interface n'hérite d'aucune méthode.

{{domxref("URLUtilsReadOnly.toString()")}}
Un {{domxref("DOMString")}} contenant l'URL entier. C'est un synonyme de {{domxref("URLUtilsReadOnly.href")}}.
