{{JSRef}}

O método clear() remove todos os elementos de um objeto Map.

Sintaxe

meuMapa.clear();

Valor retornado

{{jsxref("undefined")}}.

Exemplos

Usando o método clear

var meuMapa = new Map();
meuMapa.set('bar', 'baz');
meuMapa.set(1, 'foo');

meuMapa.size;       // 2
meuMapa.has('bar'); // true

meuMapa.clear();

meuMapa.size;       // 0
meuMapa.has('bar')  // false
