Window.onuserproxymity 属性代表一个 {{domxref("EventHandler")}}, 当触发 {{event("userproximity")}} 事件时会调用这个函数。这些事件是 {{domxref("UserProximityEvent")}} 类型的，在设备传感器检测到对象变得靠近时触发。

语法

window.onuserproximity = eventHandler
