概述

返回所使用浏览器的内部名称.

语法

codeName = window.navigator.appCodeName

参数

  • codeName 是一个字符串,表示浏览器内部名称.

例子

dump(window.navigator.appCodeName);

笔记

Mozilla, Netscape 6, 和 IE5 的内部名称都是 "Mozilla".
