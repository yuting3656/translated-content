{{ APIRef("HTML DOM") }}{{SeeCompatTable}}

强制对HTML元素进行拼写和语法检查，即使用户没有关注元素。此方法将覆盖浏览器的行为。检查的界面，例如是否出现红色下划线，由浏览器确定。

Syntax

element.forceSpellCheck()
