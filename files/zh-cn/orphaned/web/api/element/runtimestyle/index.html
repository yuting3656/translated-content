概要

Element.runtimeStyle 是一个元素专有属性，和 {{domxref("HTMLElement.style")}} 相似，除了其中的样式属性外，{{domxref("HTMLElement.style")}} 具有更高的优先级和修改能力。runtimeStyle 不能修改 style 中的content属性，其在旧版的IE浏览器上可用。

规范

不属于任何规范的一部分。

微软对此有一篇描述文档 has a description on MSDN.
