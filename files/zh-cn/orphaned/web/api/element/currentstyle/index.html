概述

Element.currentStyle 是一个与 {{domxref("window.getComputedStyle")}}方法功能相同的属性。这个属性实现在旧版本的IE浏览器中.

规范

没有相关规范。

Microsoft 在MSDN中对该属性进行了描述.
