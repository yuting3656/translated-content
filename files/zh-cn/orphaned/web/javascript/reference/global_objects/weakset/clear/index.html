clear() 方法用于删除 WeakSet 对象的所有元素，但是已经不是 ECMAScript 的一部分了。

语法

ws.clear();

示例

使用 clear方法

var ws = new WeakSet();

ws.add(window);
ws.has(window);  // true

ws.clear();

ws.has(window); // false

规范

没有规范或草案。该方法原本计划包括在 ECMAScript 6，但是在草案 revision 28 (October 14, 2014) 被抛弃了。浏览器原先的实现不久后也被移除了，它从来不是标准的一分子。
