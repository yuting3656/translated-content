错误信息

TypeError: null has no properties
TypeError: undefined has no properties

错误类型

{{jsxref("TypeError")}}.

哪里出错了?

 {{jsxref("null")}} 和 {{jsxref("undefined")}}中，没有你需要的属性。

例子

null.foo;
// 错误类型: null没有这个属性

undefined.bar;
// 错误类型: undefined没有这个属性
