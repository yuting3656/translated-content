min プロパティは、センチメートル単位でセンサーが報告可能な距離を提供します。

構文

var value = instanceOfDeviceProximityEvent.min;

値

デバイスの近接度センサーが報告可能なセンチメートル単位の最小距離を表す正の数値です。
