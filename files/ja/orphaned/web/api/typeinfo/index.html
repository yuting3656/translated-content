廃止された TypeInfo インターフェイスは、ドキュメントのスキーマ内の名前空間に名前をキー入力することによって DOM データ型を記述するために使用することを目的としていました。

プロパティ

{{domxref("TypeInfo.typeName")}} {{Readonlyinline}}
型の名前を示す {{domxref("DOMString")}} を返します。
{{domxref("TypeInfo.typeNamespace")}} {{Readonlyinline}}
スキーマ内の型の名前空間を示す {{domxref("DOMString")}} を返します。

メソッド

{{domxref("TypeInfo.isDerivedFrom()")}}
{{jsxref("Boolean")}}. を返します。

定数

定数 値
DERIVATION_RESTRICTION 1
DERIVATION_EXTENSION 2
DERIVATION_UNION 4
DERIVATION_LIST 8
