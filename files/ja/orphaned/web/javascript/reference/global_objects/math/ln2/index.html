Math.LN2 プロパティは2の自然対数、およそ0.693を表します。

Math.LN2=ln(2)≈0.693\mathtt{\mi{Math.LN2}} = \ln(2) \approx 0.693

解説

LN2 は Math オブジェクトの静的プロパティなので、 Math オブジェクトを生成してプロパティとして使用するのではなく、常に Math.LN2 として使用するようにしてください (Math はコンストラクターではありません)。

例

Math.LN2 の使用

以下の関数は、2の自然対数を返します。

function getNatLog2() {
  return Math.LN2;
}

getNatLog2(); // 0.6931471805599453
