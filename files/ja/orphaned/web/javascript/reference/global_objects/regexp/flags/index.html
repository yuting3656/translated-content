解説

flags プロパティのフラグはアルファベット順にソートされます(左から右へ 例えば、"gimuy")。

例

flags の使用

/foo/ig.flags;   // "gi"
/bar/myu.flags;  // "muy"

ポリフィル

if (RegExp.prototype.flags === undefined) {
  Object.defineProperty(RegExp.prototype, 'flags', {
    configurable: true,
    get: function() {
      return this.toString().match(/[gimsuy]*$/)[0];
    }
  });
}
