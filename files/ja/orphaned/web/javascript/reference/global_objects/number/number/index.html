Number() コンストラクターは、 {{jsxref("Number")}} オブジェクトを生成します。

構文

new Number(value)

引数

value
作成されるオブジェクトの数値による値です。

例

const a = new Number('123'); // a === 123 is false
const b = Number('123');     // b === 123 is true
a instanceof Number;         // is true
b instanceof Number;         // is false
