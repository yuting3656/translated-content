減算演算子 (-) は2つのオペランドを減算し、それらの差を生成します。

{{EmbedInteractiveExample("pages/js/expressions-subtraction.html")}}

構文

Operator: x - y

例

数値による減算

5 - 3     // 2
3 - 5     // -2

非数による減算

'foo' - 3 // NaN
