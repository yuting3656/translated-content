剰余代入演算子 (%=) は、変数を右辺のオペランドの値で除算し、剰余を変数に代入します。

{{EmbedInteractiveExample("pages/js/expressions-remainder-assignment.html")}}

構文

Operator: x %= y
Meaning:  x  = x % y

例

剰余代入の使用

// 以下の変数を想定
//  bar = 5

bar %= 2     // 1
bar %= 'foo' // NaN
bar %= 0     // NaN
