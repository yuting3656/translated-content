左シフト代入演算子 (<<=) は、指定された量のビットを左に移動し、結果を変数に代入します。

{{EmbedInteractiveExample("pages/js/expressions-left-shift-assignment.html")}}

構文

Operator: x <<= y
Meaning:  x   = x << y

例

左シフト代入の使用

let a = 5;
// 00000000000000000000000000000101

bar <<= 2; // 20
// 00000000000000000000000000010100
