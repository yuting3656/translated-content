代入（単純代入）演算子 (=) は、変数に値を代入するために使用されます。割り当て操作は、割り当てられた値を評価します。 単一の値を複数の変数に割り当てるため、代入演算子の連鎖が可能です。

{{EmbedInteractiveExample("pages/js/expressions-assignment.html")}}

構文

Operator: x = y

例

代入と連鎖

// 以下の変数を想定
//  x = 5
//  y = 10
//  z = 25

x = y     // x は 10
x = y = z // x, y そして z は全て 25
