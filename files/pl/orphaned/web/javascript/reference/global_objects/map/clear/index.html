Metoda clear() usuwa wszystkie elementy z obiektu Map.

var map1 = new Map();

map1.set('bar', 'baz');
map1.set(1, 'foo');

console.log(map1.size);
// wartość zwracana: 2

map1.clear();

console.log(map1.size);
// wartość zwracana: 0

Składnia

myMap.clear();

Zwracana wartość

{{jsxref("undefined")}}.

Przykłady

Używanie metody clear

var myMap = new Map();
myMap.set('bar', 'baz');
myMap.set(1, 'foo');

myMap.size;       // 2
myMap.has('bar'); // true

myMap.clear();

myMap.size;       // 0
myMap.has('bar')  // false
