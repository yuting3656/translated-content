Metoda add() dodaje nowy element o określonej wartości (value) na koniec obieku Set.

Składnia

mySet.add(value);

Parametry

value
Wymagany parametr. Wartość elementu, która zostanie dodana do obieku Set.

Zwracana wartość

Obiekt Set.

Przykłady

Użycie metody add

var mySet = new Set();

mySet.add(1);
mySet.add(5).add('some text'); // można stworzyć "łańcuch"

console.log(mySet);
// Set [1, 5, "some text"]
